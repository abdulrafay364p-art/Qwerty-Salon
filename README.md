# Aurelle Maison de Beauté ✨

A modern and elegant beauty salon website built with React, TypeScript, and Vite.  
The project focuses on a premium user experience with a clean design, responsive layout, and smooth performance.

## 🚀 Features

- Modern luxury salon website design
- Fully responsive (Mobile, Tablet, Desktop)
- React component-based architecture
- Type-safe development with TypeScript
- Fast development and optimized performance using Vite
- Elegant hero section
- About salon section
- Beauty services showcase
- Pricing/packages section
- Gallery section
- Customer testimonials
- Appointment booking section
- Contact information and social links
- Smooth UI animations
- SEO-friendly structure

## 💄 Services Included

- Hair Styling
- Hair Treatments
- Skincare
- Makeup Services
- Beauty Treatments
- Spa & Relaxation Services

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- HTML5
- CSS3
- JavaScript / TypeScript
- ESLint
